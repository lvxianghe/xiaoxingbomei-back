# 小型博美AI 后端

## 项目介绍

作为个人大模型应用开发学习，后端项目
”小型博美“是一个综合性AI服务平台，旨在为用户提供多种AI智能应用服务。



### 技术栈

 - jdk：17
 - springAI 1.0
 - mybatis
 - mysql
 - simple vector
 - log4j2
 - 模型接入：ollama/openai
 - 大模型：deepseek-v3、deepseek-r1、gemini2.5-pro、grok3......
